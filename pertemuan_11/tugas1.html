C. Soal Latihan / Tugas

1. Apa gunanya CSS?

CSS (Cascading Style Sheet) digunakan untuk mengatur tampilan dan tata letak halaman website, seperti warna, jenis huruf, ukuran teks, jarak, dan posisi elemen HTML. Dengan menggunakan CSS, tampilan website menjadi lebih rapi, menarik, dan mudah dikelola tanpa harus mengubah struktur HTML.

2. Berdasarkan website yang telah anda buat sebelumnya buatlah CSS dengan menggunakan Embed?

CSS dengan metode Embed adalah CSS yang dituliskan langsung di dalam tag <style> pada bagian <head> dokumen HTML. Metode ini digunakan untuk mengatur tampilan halaman web tanpa menggunakan file CSS terpisah. Contoh penerapannya adalah mengatur warna latar belakang, jenis huruf, dan ukuran teks langsung di dalam halaman HTML.

3. Setting tampilan website anda dengan CSS?

Setting tampilan website dengan CSS dilakukan dengan cara menentukan aturan style pada elemen HTML, seperti pengaturan font, warna, margin, padding, dan layout halaman. Dengan CSS, tampilan website menjadi konsisten, lebih mudah diubah, dan lebih menarik bagi pengguna.
